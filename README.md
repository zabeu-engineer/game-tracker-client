# 🎮 Game Tracker Web

A modern React web application for managing your video game collection and backlog.

This project is the **frontend client** for the Game Tracker API and allows users to interact with the backend service through a clean and simple interface.

Built to demonstrate real-world frontend architecture and integration with a RESTful backend.

Based on the **Game Tracker API** backend service.

---

# ✨ Features

- 📚 View your game collection
- ➕ Add new games
- 🗑 Delete games
- 🔎 Search games with filters
- 🎯 Track gaming progress (backlog, playing, completed)
- 🖥 Multi-platform support
- ⚡ Fast UI with React
- 🔌 REST API integration

---

# 🧠 Learning Goals

This project was created as a learning environment for frontend development and backend integration.

Concepts explored in this repository include:

- React component architecture
- Atomic Design principles
- Custom Hooks
- REST API integration
- State management
- Clean frontend structure
- Separation of concerns
- Type-safe development with TypeScript
